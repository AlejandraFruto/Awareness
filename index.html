Que acaba de ocurrir

Esta pagina forma parte de un ejercicio academico controlado de simulacion de phishing, realizado con fines pedagogicos para reforzar buenas practicas de ciberseguridad.

El sitio que visitaste imitaba un portal institucional. Aunque el formulario parecia autentico, el objetivo es educativo: ayudarte a reconocer este tipo de ataques antes de que ocurran en un contexto real.

El phishing es una tecnica de ingenieria social usada para robar credenciales, acceder a sistemas privados o instalar software malicioso. Identificarlo a tiempo ayuda a proteger tus cuentas y a tu organizacion.

Como reconocer un ataque de phishing

Presta atencion a estas senales de alerta:

  • La URL no coincide exactamente con el dominio oficial.
  • El correo o enlace viene de un remitente desconocido o sospechoso.
  • El sitio no usa HTTPS o el certificado no parece valido.
  • El mensaje usa urgencia, amenazas o presion para que actues rapido.
  • Hay errores de ortografia, imagenes de baja calidad o elementos desalineados.
