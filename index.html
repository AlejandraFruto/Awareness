Qué acaba de ocurrir

Esta página forma parte de un ejercicio académico controlado de simulación de phishing, realizado con fines pedagógicos para reforzar buenas prácticas de ciberseguridad.

El sitio que visitaste imitaba un portal institucional. Aunque el formulario parecía auténtico, el objetivo es educativo: ayudarte a reconocer este tipo de ataques antes de que ocurran en un contexto real.

El phishing es una técnica de ingeniería social usada para robar credenciales, acceder a sistemas privados o instalar software malicioso. Identificarlo a tiempo ayuda a proteger tus cuentas y a tu organización.

Cómo reconocer un ataque de phishing

Presta atención a estas señales de alerta:

  • La URL no coincide exactamente con el dominio oficial.
  • El correo o enlace viene de un remitente desconocido o sospechoso.
  • El sitio no usa HTTPS o el certificado no parece válido.
  • El mensaje usa urgencia, amenazas o presión para que actúes rápido.
  • Hay errores de ortografía, imágenes de baja calidad o elementos desalineados.
